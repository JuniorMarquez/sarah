Boss - Refine Search

In back end, go to Admin CP / Extensions / Modules / Boss - Refine Search > Edit

Add Filters

Click on "Add Filter" tab, then click on "Add" button (top right corner)   to add a new filter. You will see the interface same as the Opencart filters except the filter Image.

Add a filter to a product

Go to Admin CP / Catalog / Products > Edit your product -> Links tab -> Filters -> Add your filters. More detail here http://docs.opencart.com/en-gb/catalog/product/link/

Module Setting

  • 1. Image Module Size(WxH): The width and height of filter images.
  • 2. Display: Display filter by text or image.
  • 3. Show under product: Show the value of filter in a product layout.
  • 4. Status: You can disable or enable this extension
