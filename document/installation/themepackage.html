Install Theme Package

If your website goes to live, you should use this method to protect your data. This tutorial will help you to install the theme without lose your data.

Step 1: Install Opencart default v3.0.x to your webserver. Follow this tutorial http://docs.opencart.com/en-gb/installation/

Step 2: Find and extract sarah-home-v3.0-r4.zip in our theme package. You will see sarah-home-v30 folder and sample-database.sql.

Step 3: Upload all files in sarah-home-v30/upload folder to your webserver.

Step 4: Go to Admin CP / Dashboard / Setting button    (top right corner) -> Theme -> Refesh to clear Twig cache.

Step 5: Go to Admin CP / System / Users / User Groups / Administrator -> Edit -> Select All permission -> Save

 

 

Step 6: Install Theme Manager

- Go to Admin CP / Extensions / Extensions / Modules / Boss - Theme Manager -> Install

Step 7: Install OCMod

- Go to Admin CP / Extensions / Boss - Theme Manager (Admin CP / Extensions / Extensions / Modules / Boss - Theme Manager -> Edit)

- Go to XML OCMod -> Upload all XML files in sarah-home-v30/upload/modification_xml

- Refresh the OCMod when upload completely

 

Step 8: Install Sample Data

- Go to Admin CP / Extensions / Boss - Theme Manager / Sample Data -> Import all modules sample data.

Step 9: Install Theme

- Go to Admin CP / Extensions / Extensions / Themes / Sarah Market > Edit

- Go to General section / Status > Select Enabled > Save

Step 10: Enable theme to website

- Go to Admin CP / System / Settings / Store > Edit

- Go to General tab / Theme section > Select Sarah Market > Save
