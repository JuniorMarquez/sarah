Boss - Tags Cloud

In back end, go to Admin CP / Extensions / Modules / Boss - Tag Cloud > Edit

  • 1. Module Name: Write a name for this module
  • 2. Title: Name of this module will appear in the front end. You can leave it empty if you don't need a title.
  • 3. Limit: Limit the number of tags shown in front end.
  • 4. Min Size: Minimum size of the text
  • 5. Max Size: Maximum size of the text
  • 6. Font Weight: Font weight of the text
  • 7. Status: Disable/Enable the module
