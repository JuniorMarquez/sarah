Boss - Revo Slider

Slider Revolution is a fully developed slide in system to offer the capability to show image, videos and captions paired with simple, modern and fancy 3D transitions. On top of that, Slider Revolution is fully responsive and mobile optimized and can be displayed on any dimensions.

In back end, go to Extensions / Modules / Boss - Revolution Slider > Edit

Module Setting

  • 1. Name Name of this sliders
  • 2. Number of Slides Total slide of this slider
  • 3. Status Show status of this slider (Enable or Disable)
  • 4. Edit Go to page edit slider. Edit each slide includes: image background, caption and other option ...
  • 5. Setting Go to page slider setting: Setting all option of slider: slider width, status, name...

Slider Setting

You can config all option of each slide at here: time delay, transitions, full/box...

Config Slider

  • 1. Name Link of image background of each slide
  • 2. Status Show status of each slide (enable or disable)
  • 3. Action Include: copy, delete or edit each slide
  • 4. Drag and Drop You can drag and drop it to change position.
  • 5. Slider Setting Go to page setting for this slider
  • 6. Add new slide Add new slide for this slider

Edit Each Slide

Change Background

In General Slide Setting section, at Background select type Image, at Image click on the preview image to select a image from library, then click on the "Apply background preview" button.

Add a Caption

Click on "Add text caption" button (bottom left corner)   to add a text caption. You will see the Caption Setting section.

  • 1. Text is the text value you want to show in the front-end.
  • 2. Data x is a hozizontal position.
  • 3. Data y is a vertical position.
  • 4. Class CSS is used to change the font size and color of caption.

You can drag a Caption in preview section to change the position of this caption.

Remove a Caption

In preview section, click on a caption, then click on the "Delete Caption" button  to remove this caption.
