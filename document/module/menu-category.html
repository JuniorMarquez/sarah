Boss - Menu Category

In back end, go to Extensions / Modules / Boss - Menu Category > Edit

  • 1. Name: Write a name for this module
  • 2. Status: You can disable or enable this extension
  • 3. Title: This is the title of this module that will appear in the front end. You can leave it empty if you don't need a title.
  • 4. Title: Menu title.
  • 5. Icon: This is the icon of this module that will appear near by the title.
  • 6. Category: The category you want to display.
  • 7. Sub Category: Display children categories.
  • 8. Submenu Width: The width of submenu.
  • 9. Column: The number of submenu column.
  • 10. Background: The image background of submenu.
  • 11. Sort Order: The sort order of menu.
  • 12. Action: Remove or Add new menu.
