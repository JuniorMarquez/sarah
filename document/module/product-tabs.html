Boss - Product Tabs

Boss - Product Tabs module allows you to display products according to popular, best seller, special, latest products or choose product by name.

In the back-end, go to Admin CP / Extensions / Extensions / Modules / Boss - Product Tabs > Edit.

  • 1. General: Almost setting will be display in this tab.
  • 2. Tabs: Add or remove a tab, which will be displayed in front-end page.
  • 3. Slide: Slide function setting.
  • 4. Custom: Entry your custom content, it will be displayed in fron-end page.
  • 5. Name: Module name.
  • 6. Status: Turn on or turn off the module.
  • 7. Image (W x H): The size of products image.
  • 8. Description length: The maximum length of products description. Set 0 to hide the description. In some themes, depending on design, the product description can be removed.
  • 9. Limit: The maximum number of products.
  • 10. Column: Number of column of products.
  • 11. Product Layout: The product layout (type list or grid) on the module.
  • 12. Style: Style of the module.
  • 13. Skin: Depending on design, the skin will effect to border color, text color or background color.
  • 14. Title: A custom title will be displayed in the front-end page.
  • 15. Description: A short description will be displayed in the front-end page.

Tabs

  • Title: A tab title, it will be displayed in the front-end page.
  • Get product from: The filter products.
  • Sort Order: The number of sort order, lower number first.

Slide

  • Slidable: Turn on or turn off the slide function.
  • Number Row: The number of row will be displayed.
  • Product per row: The number of products each row.
  • Navigation: Show or hide the previous navigation and next navigation.
  • Navigation Position: The position of navigation will be placed.
  • Navigation State:
    • Visible: The navigation always shows.
    • Invisible: The navigation will be shows when hover on the module.
  • Pagination: Show or hide the panigation.
  • Center Mode: Disable or enable center function.

Custom

  • Status: Show or hide the custom description.
  • Content: The content of custom description.
