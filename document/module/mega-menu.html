Boss - Mega Menu

Boss - Mega Menu module allows you to arrange many items of header menu in different columns. Customers can easily receive a lot of information and image only by hovering your header menu.

In back end, go to Extensions / Modules / Boss - Megamenu > Edit

  • 1. Module Name: Write a name for this module
  • 2. Status Disable/Enable this module
  • 3. Edit Edit content of menu, menu row or menu column
  • 4. Delete Delete menu, menu row or menu column
  • 5. Menu column A menu row has many menu columns. A menu column has many types content as: html, product, category, information, manufacturer
  • 6. Add menu column Add a new menu column.
  • 7. Add menu row Add a new menu row. Each menu row is in one menu
  • 8. Add menu Button add new menu. A menu includes many menu rows and a menu row has many menu columns
