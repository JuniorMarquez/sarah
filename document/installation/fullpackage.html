Install Full Demo

If you install a new website, you can use this method to install this theme quickly. If your website goes to live, just only install theme package to protect your data.

Step 1: Install Opencart default v3.0.x to your web server. Follow this tutorial http://docs.opencart.com/en-gb/installation/

Step 2: Find and extract sarah-home-v3.0-r4.zip in our theme package. You will see sarah-home-v30 folder and sample-database.sql.

Step 3: Upload all files in sarah-home-v30/upload folder to your webserver.

Step 4: Go to phpMyAdmin page of your server, then import the file sample-database.sql to your database.

Notice: If you install the Opencart with custom prefix, you need to edit the file sample-database.sql first.

Find "`oc_" and replace all with "`your_custom_prefix_"

Step 5: Go to Admin CP / Dashboard / Setting button   (top right corner) -> Theme -> Refesh to clear Twig cache.

Step 6: Go to Admin CP / Extensions / Modifications -> Refresh  

Step 7: Go to Admin CP / System / Users / User Groups / Administrator -> Edit -> Select All permission -> Save
