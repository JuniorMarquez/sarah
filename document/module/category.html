Boss - Category

In the back-end, go to Admin CP / Extensions / Extensions / Modules / Boss - Category -> Edit.

  • Module Name: Is name of this modules.
  • Custom Title: The title will be show in font-end page.
  • Status: To turn on or turn off your module.
