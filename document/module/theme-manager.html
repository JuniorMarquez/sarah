Boss - Theme Manager

In your back-end, go to Admin CP / Extensions / Extensions / Modules / Boss - Theme Manager -> Edit. This module will help you to customize the front-end of your website easily.

General Setting

General tab

- Loading page is the page will cover your website when your site is loading and rendering. After your website is loaded, this page will be removed to show your complete website.

- Animation: If you enable it, the wow.js will be enable in your website. You can find out wow.js here http://mynameismatthieu.com/WOW/

- Sticky Menu: The main menu will be stick at the top of page when you scrolldown over it.

- Button to Top: This button will appear when you scroll down of page.

Product tab

- Sharing Button is the button in product detail page.

- Display Description: You have two options (tap and collapse) for disply the description of product in product detail page.

- Related Product: Show or hide the related products in product detail page.

Category tab

- Category information: Display the category information like tile, decription in category page.

- Category refine search: Display the child category in category page.

- Category list layout: The default layout of products will be show in category page.

Font & Color

This section will help you change the font and the color easier. You don't need to edit a CSS rules to change them.

How can I add a new rules?

Step 1: Click on the add  button you will see the popup.

- Type is a type of your setting. You have four options.

  • Color is used for background-color, border-color, color CSS rules.
  • Image is used for background-image CSS rules.
  • Font is used for font-family CSS rules.
  • Select is used for various situation.

- Selector is the Class name, ID name or HTML DOM of a HTML element.

- Property is a CSS properties (background-color, border-color, color, font-family,..etc), you can look for more properties here https://www.w3schools.com/cssref/default.asp

- Value is default value you want to set, you can leave it empty.

- Keyword must be unique, lowercase, no white space and no special character.

- Reference is the value for Font type and Select type, this is a one-dimensional array JSON encode.

Step 2: Click on button "Add New" after you entry completely the value.

Step 3: After add your custom rules, click on button save  (top right corner of the control frame) to save your work.
