Boss - Dynamic HTML

In back end, go to Admin CP / Extensions / Modules / Boss - Instagram > Edit

  • 1. Module Name: Write a name for this module.
  • 2. CSS Editor: Write css script for this module.
  • 3. Graphice mode: Graphical user interface.
  • 4. Code mode: HTML code editor.
  • 5. Paste: Copy from first language tab and paste to this language tab.
  • 6. Content: Content frame.
  • 7. Editor: Click on text block or image block to edit it.
  • 8. Status: Enable or disbale this module.

Edit Content

Click on the text or a image, the Editor will be displayed. You can edit the content easily.

If you want to insert the image, click on "Image" icon  on the tools bar of editor. You will see the popup.

Click on the "File Browser" icon  to insert a image from the library.

Customize HTML

Click on "Code" symbol  to switch to Code Mode, now you can edit the HTML code. After finish your code, switch to "Eye" symbol  to switch to Graphice Mode.

Customize CSS

Click on "CSS" tab to entry your CSS rules.
